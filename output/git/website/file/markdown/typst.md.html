     15 But as it turned out there has been a simpler tool for the job. And it is called Typst.
     16 
     17 Why it is simpler:
     18 
     19 - It is FOSS
     20 - Ready to write: no `\document{begin}`
     21 - Markdown based syntax where it can: eg. headings with `=`
     22 - Easy math syntax
     23 - Simple coding constructs instead of single use case based macros
     24 - FOSS templates ready to clone
     25 - Renders preview from buffer: instant update without having to safe
