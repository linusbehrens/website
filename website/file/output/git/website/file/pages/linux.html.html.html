     21 <a href="#l2" class="line" id="l2">      2</a> &lt;blockquote&gt;
     22 <a href="#l3" class="line" id="l3">      3</a> &lt;p&gt;This is the Unix philosophy: Write programs that do one thing and do
     23 <a href="#l4" class="line" id="l4">      4</a> it well. Write programs to work together. Write programs to handle text
     24 <a href="#l5" class="line" id="l5">      5</a> streams, because that is a univeRsal interface&lt;/p&gt;
     25 <a href="#l6" class="line" id="l6">      6</a> &lt;p&gt;&lt;em&gt;- Doug McIlroy, the inventor of Unix pipes and one of the
     26 <a href="#l7" class="line" id="l7">      7</a> founders of the Unix tradition&lt;/em&gt;&lt;/p&gt;
     27 <a href="#l8" class="line" id="l8">      8</a> &lt;/blockquote&gt;
     28 <a href="#l9" class="line" id="l9">      9</a> &lt;p&gt;Linux at its core is the idea of intercompatability born in the body
     29 <a href="#l10" class="line" id="l10">     10</a> of a kernel. What I love in Linux is not how fast and free it is. Not
     30 <a href="#l11" class="line" id="l11">     11</a> how amazingly many people contribute to one goal. It is the way I use it
     31 <a href="#l12" class="line" id="l12">     12</a> and how it allowed me to fit the most bare bones software to my
     32 <a href="#l13" class="line" id="l13">     13</a> needs.&lt;/p&gt;

     38 <a href="#l19" class="line" id="l19">     19</a> &lt;li&gt;&lt;a href=&quot;posts/tex&quot;&gt;&lt;/a&gt; for long-term sheets&lt;/li&gt;
