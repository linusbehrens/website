    730 </a><a href="#h12-0-49" id="h12-0-49" class="i">+&lt;p&gt;This is the Unix philosophy: Write programs that do one thing and do
    731 </a><a href="#h12-0-50" id="h12-0-50" class="i">+it well. Write programs to work together. Write programs to handle text

    760 </a><a href="#h12-0-79" id="h12-0-79" class="i">+&lt;h2 id=&quot;intercompatability&quot;&gt;Intercompatability&lt;/h2&gt;
    761 </a><a href="#h12-0-80" id="h12-0-80" class="i">+&lt;p&gt;My shell runs in &lt;a
